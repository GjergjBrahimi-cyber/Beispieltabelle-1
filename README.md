# Übung: Die Arten von Tabellen
In diesem Projekt habe ich 3 verschiedene Ansätze zur Arbeit mit Tabellen in HTML umgesetzt, um den Unterschied zwischen reiner Struktur und modernem Design zu verdeutlichen.

## Inhalt der Übung

### 1. Die Einfache HTML-Tabelle
* Fokus auf die sematische Struktur
('<table>' , '<thread>' , '<body>')

### 2. Die gestalete CSS-Tabelle (Styled Table)
* Einsatz von 'border-collapse' und Farben.

## Verwendete Technololgien
* **HTML5**
* **CSS3**
